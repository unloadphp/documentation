## Geo Whitelist

Define a whitelist of countries allowed to use an application. **Accepts an array of country names.**

```yaml
firewall:
  geo-whitelist:
    - United States
```

## Geo Blacklist

Define a blacklist of countries allowed to use an application. **Accepts an array of country names.**

```yaml
firewall:
  geo-blacklist:
    - United States
```
